# Osu-direct-Linker
Chrome extension that adds Osu!direct button in beatmap search page. OSU SUPPORT IS REQUIRED FOR IT TO WORK!!! (i guess)
Requires Osu api v1 key, for conveting beatmapsetid to beatmapid.
